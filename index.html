Computer parts

Discover the different parts of the computer

Central Processing Unit (CPU)

Often referred to as the brain of the computer, the CPU executes instructions from programs and performs basic arithmetic, logic, control, and input/output (I/O) operations specified by the instructions.

Motherboard

is a printed circuit board (PCB) that houses various components of the computer, including the CPU, RAM, expansion slots, and connectors for peripherals.

Random Access Memory (RAM)

is a type of computer memory that stores data and machine code currently being used by the CPU. It allows for quick read and write operations, enabling faster access to data than storage devices like hard drives.

Storage Devices

These include hard disk drives (HDDs), solid-state drives (SSDs), and other storage media used to store data and programs on the computer. HDDs use magnetic storage, while SSDs use flash memory for faster access times.

Power Supply Unit (PSU)

converts electrical power from an outlet into usable power for the computer's components. It provides different voltages and currents required by various parts of the computer.

Graphics Processing Unit (GPU)

Also known as a video card, the GPU is responsible for rendering images, videos, and animations on the computer monitor. It offloads graphical processing tasks from the CPU, enhancing graphics performance.

Input Devices

These include the keyboard, mouse, touchpad, and other devices used to input data and commands into the computer.

Output Devices

These include the monitor, printer, speakers, and other devices used to display information or output data from the computer.

Cooling Systems

Cooling systems such as fans, heat sinks, and liquid cooling systems are used to dissipate heat generated by the CPU, GPU, and other components to prevent overheating and ensure optimal performance.

Expansion Cards

These are additional circuit boards that can be installed into expansion slots on the motherboard to add functionality such as sound cards, network adapters, and graphics cards.
